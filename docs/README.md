## Overview

This repository contains two complementary documents related to the GC-IMS curve harmonization workflow.
Each document serves a specific purpose within the methodology: one focuses on the **definition and implementation of the harmonization function**, and the other focuses on the **application of the method to individual samples and the corresponding error estimation**.

---

## **1. harmonization**

[harmonization.md](https://github.com/tecladuran/gcims-curve-harmonization/blob/main/docs/harmonization.md)


This document describes the **core harmonization procedure**, including:

* The mathematical formulation of the harmonization problem.
* The definition and implementation of the function used to optimize **shift** and **scale** between calibrations.
* Description of the optimization strategy and constraints.
* Visual validation of the harmonization using example datasets.
* Reference implementation ready to be sourced and reused in other analyses.

This section is the central technical description of the harmonization algorithm.

---

## **2. methodology**
[methodology.md](https://github.com/tecladuran/gcims-curve-harmonization/blob/main/docs/methodology.md)

This document details the **application workflow** using real GC-IMS data from multiple samples.
It contains:

* Pre-processing of individual datasets.
* Error analysis using scaled and unscaled calibration references.
